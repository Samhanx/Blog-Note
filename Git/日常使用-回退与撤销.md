## 日常使用 - 回退与撤销

```bash
# 撤回所有已经添加到暂存取的修改
git reset HEAD
# 撤回已经添加到暂存区某文件，这个命令在添加暂存区后 git status 就有提示
git reset HEAD <file>

# 撤回提交，保留工作区的所有修改，撤回暂存区的所有修改
git reset HEAD^
# 等同于
git reset --mixed HEAD^
# 等同于
git reset HEAD~1
# 从当前开始向上撤回 n 个提交
git reset HEAD~n

# 软撤回提交，保留工作区的所有修改，保留暂存区的所有修改
git reset --soft HEAD^

# 硬撤回提交，删除工作区的所有修改，撤回暂存区的所有修改，相当于完全回到之前提交的状态
git reset --hard HEAD^
```
